language_html_data_active_event_handler_body_throw

If application data is used directly as an inline event-handler body, this throws the standard canary Error when that event fires. It is useful for test harnesses that can trigger the target event and observe uncaught page errors. The value performs no action before the surrounding event is dispatched.
