real_world_invalid_url_backslash_scheme_separators

Reverse solidus is not an RFC URI delimiter, but WHATWG special-URL parsing treats these backslashes as slashes and repairs the input to `https://example.com/path`. A validator and fetcher using different URL dialects can therefore disagree on every component boundary. RFC 3986 excludes raw backslash, while WHATWG special authority parsing handles it as a validation-error separator: https://www.rfc-editor.org/rfc/rfc3986.html#section-2 https://url.spec.whatwg.org/#special-authority-ignore-slashes-state
