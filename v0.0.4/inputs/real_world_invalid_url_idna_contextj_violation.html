real_world_invalid_url_idna_contextj_violation

U+200D ZERO WIDTH JOINER occurs between Latin letters without the contextual conditions required by IDNA. It is visually hidden, so deleting it can make the invalid source appear to be the valid `ab.example` host. RFC 5892 defines CONTEXTJ requirements for U+200D, which domain-to-ASCII processing enforces: https://www.rfc-editor.org/rfc/rfc5892.html#appendix-A.2
