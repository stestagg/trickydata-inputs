utf8_chunk_ascii_a

A single ASCII letter is valid UTF-8 on its own. When appended to the paired truncated EURO SIGN prefix, the concatenation E2 82 41 is invalid because ASCII 0x41 appears where a continuation byte is required. This adjunct tests validators that join independently supplied chunks. See the Unicode well-formed UTF-8 byte sequences in Table 3-7: https://www.unicode.org/versions/Unicode15.0.0/ch03.pdf
