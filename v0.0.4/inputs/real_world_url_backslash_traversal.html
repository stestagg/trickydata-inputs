real_world_url_backslash_traversal

WHATWG special URLs treat backslash as a path separator, and Windows filesystems do too. An RFC-only filter may see ordinary path data while a browser, proxy, or filesystem resolves `..\` as a parent segment. The WHATWG URL parser treats reverse solidus like slash for special schemes: https://url.spec.whatwg.org/#url-path-state
