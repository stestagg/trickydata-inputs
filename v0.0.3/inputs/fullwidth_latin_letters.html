fullwidth_latin_letters

This is the word "HELLO" written with FULLWIDTH LATIN CAPITAL LETTERS (U+FF28 U+FF25 U+FF2C U+FF2C U+FF2F). Each is a single code point and grapheme, but its East Asian Width property is Fullwidth, so a monospace terminal or fixed-cell grid allocates two columns per character. Code that measures the string by code-point or grapheme count assumes five columns and under-counts the real width by half, misaligning tables, padding, and truncation. Unicode Standard Annex #11 defines the East Asian Width property: https://www.unicode.org/reports/tr11/
