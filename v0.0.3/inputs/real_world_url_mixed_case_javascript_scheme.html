real_world_url_mixed_case_javascript_scheme

URI schemes are case-insensitive, so this is a `javascript:` URL, not a network fetch merely because it contains host-shaped authority text. Its script body is a harmless line comment. Legacy Node.js `url.parse()` could report a spoofed hostname when the scheme used mixed case, leading applications to make incorrect security decisions. Node.js published this as CVE-2018-12123: https://nodejs.org/en/blog/vulnerability/november-2018-security-releases
