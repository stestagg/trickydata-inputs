language_html_data_active_close_script_throw

Inside an inline script this closes the script at the HTML tokenizer layer, inserts an SVG load handler that throws the standard canary error, and opens a replacement script element. JavaScript string escaping alone does not prevent this HTML context break. The thrown error is the only intended runtime side effect.
