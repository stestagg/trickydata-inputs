soft_hyphen_in_word

This is the word "automobile" with a SOFT HYPHEN (U+00AD) inserted after "auto". A soft hyphen marks a permitted hyphenation point: it renders as nothing when the word fits on the line, but appears as a visible hyphen glyph if the renderer breaks the line there. The same bytes therefore render with different width and a different glyph count depending on the surrounding layout, and copying the displayed text can silently include or drop the character. It tests conditional rendering and width that depends on line breaking rather than on the text alone. Unicode Standard Annex #14 describes SOFT HYPHEN line-break behaviour: https://www.unicode.org/reports/tr14/
