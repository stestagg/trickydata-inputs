integer_u32_msvc_new_fill

This repeated 0xcd byte is used by the Microsoft CRT debug heap for newly allocated objects. Seeing it as an integer is a strong canary for uninitialized, stale, or out-of-bounds data crossing a typed boundary. Microsoft documents the debug-heap fill values here: https://learn.microsoft.com/en-us/cpp/c-runtime-library/crt-debug-heap-details
