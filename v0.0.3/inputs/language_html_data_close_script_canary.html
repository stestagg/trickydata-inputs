language_html_data_close_script_canary

The HTML tokenizer recognizes a script end tag before JavaScript string parsing. Unescaped data inside an inline script can therefore close the script and create this inert custom element even when JavaScript quoting appears correct. Script-data tokenization is defined here: https://html.spec.whatwg.org/multipage/parsing.html#script-data-state
