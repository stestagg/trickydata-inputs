real_world_url_repeated_slash_path

The doubled slash creates an empty path segment and is not generically equal to the paired path. Servers and proxies often merge slashes anyway, which can make access-control matching disagree with request routing. Apache tracked unexpected URL matching with `MergeSlashes OFF` as CVE-2021-30641, demonstrating that repeated separators affect real security boundaries: https://httpd.apache.org/security/vulnerabilities_24.html#CVE-2021-30641
