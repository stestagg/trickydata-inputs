combining_mark_overflow

This is a single base letter "e" (U+0065) followed by thirty COMBINING DIACRITICAL MARKS drawn from the U+0300 block, alternating above and below the baseline. All thirty-one code points form one grapheme cluster, yet the stacked marks grow far taller than the font's line box and bleed into the lines above and below — the "Zalgo" effect — so naive layout that assumes a glyph stays within its em box will overlap neighbouring rows. It also stresses cluster segmentation, cursor movement, and combining-mark normalisation. Unicode Standard Annex #29 keeps a base and its trailing combining marks in a single grapheme cluster: https://www.unicode.org/reports/tr29/
