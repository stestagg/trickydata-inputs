invalid_utf8_cesu8_surrogate_pair

This is the CESU-8 representation of GRINNING FACE (U+1F600): its UTF-16 surrogate pair is encoded as two three-byte sequences. Standard UTF-8 must encode U+1F600 directly as F0 9F 98 80 and must reject both surrogate sequences. This catches accidental CESU-8 or Java Modified UTF-8 acceptance. See Unicode Technical Report #26 on CESU-8: https://www.unicode.org/reports/tr26/
