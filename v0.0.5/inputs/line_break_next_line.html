This input separates "A" and "B" with NEXT LINE (U+0085). It is a Unicode whitespace and line-boundary character encoded as two UTF-8 bytes, but many line-oriented APIs recognize only ASCII LF or CRLF. Unicode Standard Annex #14 assigns U+0085 the Next Line line-break class: https://www.unicode.org/reports/tr14/
