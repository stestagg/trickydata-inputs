real_world_url_ideographic_dot_host

The visible separator is U+3002 IDEOGRAPHIC FULL STOP rather than ASCII dot. UTS #46 domain processing maps it to U+002E, so an IDNA-aware URL parser produces the paired ASCII host. Byte-level suffix checks can miss it. UTS #46 defines processing of mapped domain code points: https://www.unicode.org/reports/tr46/#Processing
