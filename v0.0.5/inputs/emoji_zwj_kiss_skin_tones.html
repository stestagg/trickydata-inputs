emoji_zwj_kiss_skin_tones

This is the RGI emoji ZWJ sequence "kiss: woman, man, medium-light skin tone, dark skin tone": WOMAN (U+1F469) + EMOJI MODIFIER FITZPATRICK TYPE-3 (U+1F3FC), ZERO WIDTH JOINER (U+200D), HEAVY BLACK HEART (U+2764) + VARIATION SELECTOR-16 (U+FE0F), ZWJ, KISS MARK (U+1F48B), ZWJ, MAN (U+1F468) + EMOJI MODIFIER FITZPATRICK TYPE-6 (U+1F3FF). Ten code points collapse into a single glyph occupying roughly two columns when the font carries the sequence, but a font that lacks it falls back to rendering each component separately, so the same bytes occupy anywhere from one to seven-plus emoji of width. It exercises width estimation, fallback rendering, and the most-scalars-per-glyph corner. Unicode Technical Standard #51 defines emoji ZWJ sequences and skin-tone modifiers: https://www.unicode.org/reports/tr51/#Emoji_ZWJ_Sequences
